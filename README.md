# interview-hack

Hack Interview is a program in which you 
can answer questions
asked in programming interviews.

The program can help you improve
your interview skills in the workplace.


Steps to install it:
1. Download the file main.py.
2. From the powershell terminal enter the file directory.
3. Run the file with: python main.py.


Make sure you have Python installed
on your computer. Thank you.
